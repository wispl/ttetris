# ttetris

Tetris in the terminal

### Features

* Tetris, in the terminal
* SRS rotation
* Holding
* Previews
* Music

#### Todo

#### Missing

### Building

A compiler which supports at least C99 is needed
```
git clone https://github.com/wispl/ttetris.git
cd ttetris
make
```

#### Dependencies

ncurses
miniaudio (vendored)
